# Documentation

This directory contains project documentation organized by relevance.

## Structure

### `active/` - Current Documentation
Documentation that is currently relevant and actively maintained:
- **DEPLOYMENT_STATUS.md** - Current deployment status and next steps
- **MONGODB_ATLAS_SETUP.md** - MongoDB Atlas setup guide
- **CLOUDINARY_SETUP.md** - Cloudinary integration guide
- **GALLERY_SYSTEM.md** - Gallery system documentation
- **VIDEO_PROCESSING_STRATEGY.md** - Video processing workflow
- **ADMIN_PANEL.md** - Admin panel documentation
- **DEBUG_GUIDE.md** - Debugging guide
- **TESTING_CHECKLIST.md** - Testing procedures

### `archive/` - Historical Documentation
Documentation from planning phases and completed refactoring:
- **planning/** - Planning documents and options analysis
- **analysis/** - Analysis documents and summaries

## Quick Start

For new developers, start with:
1. `active/DEPLOYMENT_STATUS.md` - Understand current state
2. `active/MONGODB_ATLAS_SETUP.md` - Database setup
3. `active/CLOUDINARY_SETUP.md` - File storage setup
